# Active Execution Plans

Agents should dump their short-term active state here when executing complex workflows to avoid context collapse. 
Always refer to this directory for active `[task-name].md` files.

Once an execution plan is complete, it should be moved to the `completed/` directory.


# 活跃执行计划
在执行复杂工作流程时，代理应将自身的短期活动状态存放在此处，以避免出现上下文混乱的情况。
请始终参考此目录来查找名为 `[任务名称].md` 的活动文件。
一旦执行计划完成，就应该将其移动到“completed/”目录中。
