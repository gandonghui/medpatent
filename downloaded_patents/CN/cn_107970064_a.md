## 📝 摘要

The invention provides a fracture reduction surgical robot system subjected to remote control and a reduction control method, and belongs to the field of surgical robots. The system comprises surgicalimage acquisition equipment, a fracture reduction robot and a remote-control workstation, wherein a graphical interface of a planned reduction path for doctors to conduct an operation is arranged onthe remote-control workstation. During work, the surgical image acquisition equipment is used for acquiring real-time perspective image of the operation and transmitting the perspective image to the remote workstation; the remote workstation is used for displaying the received image on the image interface (the graphical interface) of the planned reduction path, so that the operators (the doctors)can implement fracture reduction path planning in accordance with the image interface; the remote workstation is used for implementing computing in accordance with a path planning result, so that a robot control amount is obtained, and the remote workstation is used for transmitting the robot control amount to the fracture reduction robot; and the fracture reduction robot is used for completing corresponding motions in accordance with the received control amount. The system provided by the invention is simple in constitution and clear and intuitive in using method; and the system can respond to various complex fracture situations.

**（中文摘要）**

本发明提出种远程操控的骨折复位手术机器人系统及复位控制方法，属于手术机器人领域。该系统包括：手术图像采集设备，骨折复位机器人和远程控制工作站；其中远程控制工作站上安装有用于医生进行手术的规划复位路径的图形界面。工作时，手术图像采集设备采集手术实时的透视图像并传递到远程工作站；远程工作站将接收到的图像在规划复位路径的图像界面上显示出来，操作者根据图像界面进行骨折复位路径规划，远程工作站根据路径规划结果计算得到机器人控制量并发送给骨折复位机器人，骨折复位机器人根据接收到的控制量完成相应运动。本发明系统构成简单，使用方法清晰直观，可以应对各种复杂的骨折情况。
