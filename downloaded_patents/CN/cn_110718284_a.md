## 📝 摘要

The invention discloses an interaction method and system of three-dimensional medical image data, and the method comprises the steps: three-dimensional model data corresponding to a medical image is uploaded to a cloud server, wherein the cloud server distributes an identity recognition code for the three-dimensional model data; corresponding three-dimensional model data is downloaded to a mobileterminal according to the identity identification code, wherein the mobile terminal displays the three-dimensional model data; and the mobile terminal collects a real-time image of a surgical site ofa patient through an AR technology, and superposes the real-time image and the three-dimensional model data to obtain positioning information of a focus of the patient. In the embodiment of the invention, the transmission, display and interaction of the three-dimensional image are integrated, the display process of the medical image is simplified, more intuitive image data can be provided, a morecomprehensive image diagnosis mode is provided, and the problem of uneven medical resources in the prior art is solved.

**（中文摘要）**

本发明公开了一种三维医疗影像数据的交互方法及系统，其中，该方法包括：将医疗影像对应的三维模型数据上传至云端服务器，由所述云端服务器为所述三维模型数据分配身份识别码；根据所述身份识别码将对应的三维模型数据下载到移动终端，所述移动终端对所述三维模型数据进行展示；所述移动终端通过AR技术采集患者手术部位实时图像，将所述实时图像与所述三维模型数据进行叠加，获得患者病灶的定位信息。在本发明实施例中，把三维影像的传输、展示和交互合为一体，简化了医疗影像的展示过程，能够提供更加直观的图像数据，提供更全面的影像诊断方式，解决现有中医疗资源不平均的问题。
