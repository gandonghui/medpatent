## 📝 摘要

The embodiment of the invention provides a control method of a surgical vision follow-up system and the surgical vision follow-up system. The control method comprises the following steps that an imagecollection module continuously collects image information of a target, and sends the image information to a control module; the control module performs screening in a plurality of pieces of image information sent from the image collection module, and sends a target image to a user terminal after the target image is screened out; the user terminal feeds back space position information to the control module; and the control module generates a control instruction used for controlling the image collection module and the user terminal to do synchronous action according to the space position information fed back by the user terminal, and controls the image collection module and the user terminal to do synchronous action through the control instruction. The movement of a robot tail end microscope is controlled in a master-slave mode, so that the microscope view field movement is realized; in addition, the image in the view field range is transmitted back to a doctor; a more intuitive observation mode is obtained; and two hands of a user are liberated.

**（中文摘要）**

本发明实施例提供一种手术视觉随动系统的控制方法及手术视觉随动系统，所述控制方法包括如下步骤：图像采集模块连续采集目标的图像信息，并发送至控制模块；控制模块在图像采集模块发送的若干图像信息中进行筛选，筛选出目标图像后向用户终端发送目标图像；用户终端向控制模块反馈空间位置信息，控制模块根据用户终端反馈的空间位置信息生成用于控制图像采集模块与用户终端同步动作的控制指令，并通过控制指令控制图像采集模块与用户终端同步动作。本发明通过主从方式控制机器人末端显微镜的运动，以此实现显微镜视野移动，并将其视野范围内的图像传回给医生，获得更直观的观察方式，解放使用者的双手。
