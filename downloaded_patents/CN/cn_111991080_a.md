## 📝 摘要

The embodiment of the invention provides a method and a system. The method comprises the following steps: constructing a three-dimensional virtual model of a diseased part, capturing a real-time imageof the diseased part by using a visual positioning module, and simultaneously displaying the three-dimensional virtual model and the real-time image by adopting a display module, acquiring first positioning information of the diseased part by using the visual positioning module, moving the visual positioning module according to the first positioning information, completely overlapping the image of the diseased part in the real-time image with the three-dimensional virtual model to obtain an overlapping image, and determining the surgical entrance of a surgical instrument to the diseased partby using the overlapping image, so that more accurate and rapid operation entrance positioning is realized. A doctor can intuitively observe information such as blood vessel distribution, nerve distribution and focus space position of the diseased part, the operation risk is greatly reduced, the operation efficiency is ensured, and meanwhile, the operation success rate is improved.

**（中文摘要）**

本发明实施例提供了一种方法和系统。在本发明实施例中，构建患病部位的三维虚拟模型，采用所述视觉定位模块捕捉所述患病部位的实时图像，采用所述显示模块同时显示所述三维虚拟模型与所述实时图像，采用所述视觉定位模块采集所述患病部位的第一定位信息，根据所述第一定位信息，移动所述视觉定位模块，使所述实时图像中患病部位的影像与所述三维虚拟模型完全重叠，得到重叠图像，采用所述重叠图像，确定手术器具进入所述患病部位的手术入口，从而实现更为精确快速的手术入口定位，医生可直观地观察患病部位的血管分布、神经分布、病灶空间位置等信息，大大地降低了手术的风险，保证了手术效率的同时，提升了手术的成功率。
