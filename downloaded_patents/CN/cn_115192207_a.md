## 📝 摘要

The invention relates to a surgical robot, a main operation table and a control method and device of the main operation table. The main operation table comprises a display part and an operation part, the display part is provided with an imaging part used for final imaging, the operation part is provided with a base and an input part which is connected with the base and can move relative to the base to generate an operation instruction, and the control method comprises the steps that the postures of the base and the imaging part are obtained; calculating a first deviation between the attitude of the base and the attitude of the imaging part; judging whether the first deviation reaches a first deviation threshold value or not; when the first deviation reaches a first deviation threshold value, compensating the first deviation so that the base is basically parallel to the imaging part; after the base is substantially parallel to the imaging portion, the control input portion generates an operation instruction for controlling movement of the end instrument. The control method is beneficial for realizing intuitive control of the input part based on the imaging part.

**（中文摘要）**

本发明涉及一种手术机器人、主操作台及其控制方法、控制装置。主操作台包括显示部及操作部，显示部具有用于最终成像的成像部，操作部具有底座和与底座连接且可相对底座运动以产生操作指令的输入部，该控制方法包括：获取底座和成像部各自的姿态；计算底座的姿态与成像部的姿态之间的第一偏差；判断第一偏差是否达到第一偏差阈值；在第一偏差达到第一偏差阈值时，补偿第一偏差使底座与成像部基本平行；在底座与成像部基本平行之后，控制输入部产生用于控制末端器械运动的操作指令。该控制方法有利于实现输入部基于成像部进行直觉控制。
