## 📝 摘要

The invention discloses a virtual reality technology-based patient visual angle ophthalmologic operation simulation method in the technical field of medical assistance. The method comprises the following steps of 1, preparing materials; selecting data information of an ophthalmic patient in a database; 2, preparing instruments; preparing a virtual display image processing device for simulating an ophthalmologic operation, and simulating a virtual surgical instrument through the virtual display image processing device; the simulated ophthalmologic operation is a corneal pterygium operation; 3, performing three-dimensional construction; scanning data of a patient is imported into virtual display image processing equipment, the virtual display image processing equipment constructs an eye model according to the original scanning data, the eye model is zoomed, measured, separated and cut, and an eye structure is displayed; 4, selecting an eye model; the doctor selects the closest eye model according to the actual condition of the patient; 5, performing simulation operation; the eye model is imported into the virtual display image processing equipment, so that the patient can intuitively feel the operation process.

**（中文摘要）**

本发明公开了医疗辅助技术领域的一种基于虚拟现实技术的患者视角眼科手术模拟方法，包括以下步骤：步骤一：材料准备；在数据库中选择眼科患者的数据资料；步骤二：器械准备；准备模拟眼科手术的虚拟显示图像处理设备，通过虚拟显示图像处理设备模拟虚拟手术器械；模拟眼科手术为角膜翼状胬肉手术；步骤三：三维构建；将患者的扫描数据导入到虚拟显示图像处理设备,虚拟显示图像处理设备根据原始扫描数据进行眼部模型的构建,对眼部模型进行缩放、测量、分离和切割操作,显示眼部结构；步骤四：选择眼部模型；医生根据患者的实际情况选择最贴近的眼部模型；步骤五：模拟操作；将眼部模型导入到虚拟显示图像处理设备，患者可以直观的感受手术过程。
