## 📝 摘要

For remote operation of the surgical robotic system, a user command for the pose of the end effector is projected into a subspace that the end effector can reach. For example, a user command having six DOF is projected into five DOF subspaces. A user interface device of six DOF may be used to more intuitively control the end effector having a limited DOF relative to the user interface device based on the projection.

**（中文摘要）**

对于外科机器人系统的远程操作，用于端部执行器的位姿的用户命令被投影到该端部执行器能够到达的子空间中。例如，具有六个DOF的用户命令被投影到五个DOF子空间。六个DOF的用户接口设备可用于基于该投影更直观地控制相对于该用户接口设备具有有限DOF的该端部执行器。
