## 📝 摘要

The invention discloses a surgical hand disinfection intelligent monitoring method and device suitable for an operating room. The method comprises the following steps: (1) acquiring a real-time video data set of hand washing; (2) preprocessing the video data set, and dividing the data set into a training set, a verification set and a test set; (3) establishing an improved YOLOv8 algorithm model, and training to obtain a target detection model; (4) adding a post-processing detection head to the target detection model to enable the target detection model to directly output a detection result; and (5) post-processing the detection result. According to the result, the completion rate of all steps, the hand washing time of each step, the time of all steps and the accuracy of all people are automatically counted, and full-automatic multi-dimensional data monitoring of the surgical hand disinfection process is achieved. In addition, a user can remotely access and transmit a detection result file through a file transmission protocol, or visualize a real-time picture with a target detection result to an interactive interface, so that the detection result is more intuitive and convenient to obtain.

**（中文摘要）**

本发明公开了一种手术室适用的外科手消毒智能化监测方法和装置。该方法包括：（1）获取洗手的实时视频数据集；（2）对视频数据集进行预处理，将数据集划分为训练集、验证集和测试集；（3）建立改进型YOLOv8算法模型，训练后获得目标检测模型；（4）给目标检测模型添加后处理检测头，使其直接输出检测结果；（5）对检测结果进行后处理。根据结果自动统计全步骤的完成率、每个步骤的洗手时间、全部步骤的时间、全人次正确率，实现外科手消毒过程的全自动多维数据监测。此外使用者可以通过文件传输协议远程访问并传输检测结果文件，或将带目标检测结果的实时画面可视化到交互界面上，使检测结果的获取更加直观便利。
