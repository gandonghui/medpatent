## 📝 摘要

The invention provides an endoscope image display method and an endoscope system, and the method comprises the steps that a processor of the endoscope system receives a first mode instruction, and the first mode instruction is used for indicating to start an immersion mode; and in response to the first mode instruction, only displaying the real-time picture or only displaying the real-time picture and associated picture information in at least part of pages of the first display screen, and simultaneously displaying at least one interface element in the second display screen. Therefore, related information contents in the first display screen for displaying real-time pictures can be synchronously transferred to the second display screen according to user instructions, so that the first display screen reaches the purity degree required by a user, the second display screen displays the related information contents, the surgical field picture purity degree can be improved, the cognitive load of surgery or examination is reduced, and the user experience is improved. And the user still can intuitively obtain related information contents, so that the user experience is improved.

**（中文摘要）**

本申请提出一种内窥镜图像显示方法及内窥镜系统，方法包括：内窥镜系统的处理器接收第一模式指令，第一模式指令用于指示开启沉浸模式；响应第一模式指令，在第一显示屏的至少部分页面中仅显示实时画面或者仅显示实时画面及相关联的画面信息，在第二显示屏中同时显示至少一种界面元素。如此，使得用于显示实时画面的第一显示屏中的相关信息内容可以根据用户指令同步转移到第二显示屏，使得第一显示屏达到用户所需纯净程度，第二显示屏显示相关信息内容，有利于提高术野画面纯净度，减少手术或检查的认知负荷，用户仍然可以直观获取到相关信息内容，提高用户体验感。
