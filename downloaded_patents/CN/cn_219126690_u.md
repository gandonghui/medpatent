## 📝 摘要

The utility model relates to a vascular intervention surgical robot control end which comprises an upper shell and a lower shell which are connected with each other, and a lower shell and a stop groove which are connected with each other through a rotating shaft, a touch screen is arranged in the upper shell, the upper shell is mounted in the lower shell, and an emergency stop switch and an indicator lamp are mounted on the upper shell; a touch screen is arranged in the upper shell, a control circuit board electrically connected with the touch screen is arranged in the lower shell, and a catheter propelling simulation knob, a guide wire rotation simulation knob and a guide wire propelling simulation knob which are connected with the control circuit board are embedded in the top of the lower shell. The guide wire propulsion simulation knob is provided with an independent hysteresis damper, and a real-time force feedback system of a guide wire conveying system is combined, so that a dynamic damping feeling is provided for an operator, the resistance borne by the guide wire in the propulsion process can be more intuitively felt, and the control synchronism is improved.

**（中文摘要）**

本实用新型涉及一种血管介入手术机器人控制端，包括相互连接的上壳体和下壳体，以及通过转动轴进行相互连接的下壳体和止槽；所述上壳体内设置有触摸屏，所述上壳体安装在下壳体内，所述上壳体上安装急停开关与指示灯；所述下壳体内部安装有与触摸屏电性连接的控制电路板，所述下壳体顶部嵌设有与控制电路板连接的导管推进模拟旋钮、导丝旋转模拟旋钮及导丝推进模拟旋钮，本实用新型通过设有上壳体、触摸屏等，导采用丝推进模拟旋钮配有独立的磁滞阻尼器，结合导丝输送系统的实时力反馈系统，为操作人员提供动态的阻尼感，可更直观的感受导丝推进过程中所受到的阻力，提高了控制的同步性。
