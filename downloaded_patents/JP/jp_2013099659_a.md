## 📝 摘要

PROBLEM TO BE SOLVED: To provide a surgical system enabling more intuitive operation.SOLUTION: The surgical system (10) includes a robotic device (14) completely disposed within the patient in the patient's body cavity (16), and a user console (12) operably coupled to the robotic device (14) via a connection component. The robotic device (14) has a camera provided to associate with a device body and positioned between first and second robot arms. The user console (12) has first and second manipulator arms (26A, 26B) configured to cooperate with the first and second robot arms, and a visual display which receives images from the camera and thereby visually shows the first and second manipulator arms (26A, 26B) as they are physically connected with the first and second robot arms.
