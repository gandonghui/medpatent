## 📝 摘要

PROBLEM TO BE SOLVED: To provide a surgery assist device capable of controlling an attitude of a mechanically-controlled surgical instrument using a surgical instrument for treatment, while suppressing influence of intuitive and measurement errors.SOLUTION: The surgery assist device for controlling an attitude of a first surgical instrument inserted into a body cavity and mechanically driven, using a second surgical instrument to be inserted into the body cavity, includes: switch means that switches between a first mode of using the second surgical instrument for treatment, and a second mode using the second surgical instrument for controlling the first surgical instrument; measurement means that measures an insertion angle and insertion depth of a shaft of the second surgical instrument into the body cavity; and calculation means that, for controlling the attitude of the first instrument, determines a target position of a control object point as a control object by an operator, the point for specifying the attitude of the first surgical instrument, on the basis of a measurement result by the measurement means. The calculation means determines the target position of the control object point according to variations in the measured insertion angle and insertion depth from a start time point of the second mode.SELECTED DRAWING: Figure 3
