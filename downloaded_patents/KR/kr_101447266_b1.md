## 📝 摘要

According to a device for laparoscopic surgery of the present invention, comprised are: a case having a first joint unit and a second joint unit which are formed on both sides in the center thereof and can be bent, wherein the first joint is bent in a particular angle, and the second joint unit is bent in the same direction as the first joint unit which is forwardly and backwardly bent; a gripping unit which is coupled with the end extended from the first joint unit of the case to be able to be rotated and has a first gripping unit and a second gripping unit which are formed in the structure of a pair of scissors; a grip unit which is coupled with the end of the second joint unit of the case to be able to be rotated and has a first grip and a second grip which are coupled in the shape of a pair of scissors so as to hold or cut a surgical part in the abdominal cavity; a first wire which connects the first gripping unit of the gripping unit with the first grip so that the first grip is moved in the same direction as the moved first gripping unit; and a connecting unit which connects the gripping unit with the grip unit so that the grip unit is rotated in the same direction as the rotated gripping unit. A user grips the gripping unit so as to move the same forwardly and backwardly or rotate the same, so that the grip unit is moved or rotated in the same direction as the same and the grip unit moved in the same direction as the moving user can be easily operated. Accordingly, intuitive operation is possible, and even amateurs can easily use the same, so that laparoscopic surgery can be safely performed without medical accidents.
