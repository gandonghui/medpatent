# Patent: CN-118265497-A

## Title
Medical robot with intuitive control and control method

## Abstract
The present disclosure relates to a medical, in particular surgical, collaborative robot (1) for actuating a medical, in particular surgical, end effector (2), having: a robot base (4) as a local connection point of the robot (1); -a movable robotic arm (8) connected to the robotic base (4), the robotic arm having at least one robotic arm segment (10); an end effector (2) having an end effector axis (3) attached to a robotic arm (8); and a control unit (24) adapted to control the robotic arm (8); wherein the medical robot (1) has an input device (26) with a zero axis (30), which is adapted to detect at least one force applied to the input device (26) in at least two opposite directions transverse to the zero axis (30) as input and to be provided as control instructions of the control unit (24) for controlling the position and/or orientation of the end effector (2) and/or the robot arm segment (10), and wherein an axis (32), in particular the zero axis (30), of the input device (26) is arranged parallel, in particular coaxial, to the longitudinal axis of the end effector axis (3) and/or the robot arm segment (10). Furthermore, the present disclosure relates to a surgical assistance system, a control method and a computer readable storage medium according to the parallel claims.
